# VIX-Regression
This project focused on designing a classification model to identify VIX spike events using a combination of engineered volatility features and macroeconomic surprise indicators. A Random Forest classifier was trained with class‑balanced weighting and optimized probability thresholds to improve the detection of high‑volatility episodes. The final model achieved approximately 70 percent recall and 58 percent precision in forecasting top‑decile VIX spikes. Its performance reflected the model’s ability to capture nonlinear interactions between short‑horizon volatility regimes and macro‑surprise dynamics.
